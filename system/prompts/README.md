# Prompts

This folder stores prompt guidance for agents and skills.

Use prompt files to keep:
- tone consistent
- outputs structured
- shared business rules easy to update
- UX/UI and FE handoff simple to follow

## Prompt vs Playbook

- A prompt is an instruction for one task.
- A playbook is the full handling guide for one request type from input file to final Markdown outputs.

Current active design:
- BA Agent handles both analysis and former product-structuring work.

The prompts in this scaffold are short and easy to extend.
