# Templates

This folder stores lightweight working templates.

Use these templates when:
- drafting a BRD or FRS
- shaping a story
- sketching a wireframe
- capturing meeting notes

These templates are meant to be easy to copy, adapt, validate, and turn into Markdown output files.

Checklist templates for artifact governance are in:
- `system/templates/checklists/`

These are used to create artifact review files under:
- `projects/<project-name>/_ops/generated/<requirement-name>/artifact-reviews/`
