# Prompt Snippets

## A. Full Context Load
`Load context using profile BA-full from system/context/load-profiles.md, then process requirement <file> in project <name>.`

## B. BA Task Prompt
`Load profile BA-basic, then generate clarification, BRD, FRS, user story, acceptance criteria, and feature list for <file> in project <name>.`

## C. Review Prompt
`Load profile review, then validate output quality and consistency for requirement <file> in project <name>.`

## D. Traceability Prompt
`Load profile traceability, then update RTM and requirement traceability flow for <file> in project <name>.`

## E. Test Case Prompt
`Load profile test-case, then generate or refine test-cases.md for <file> in project <name> using AC/US/FR links.`

## F. Risk Prompt
`Load profile risk, then identify blockers, dependency risks, and severity for requirement <file> in project <name>.`

## G. Fast Minimal Prompt
`Read system/context/context-summary.md only, then process requirement <file> in project <name>.`
