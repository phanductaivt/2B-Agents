# Agents

This folder defines the main working roles in the framework.

Each agent has:
- a business role
- a clear use case
- expected inputs and outputs
- playbook usage guidance
- collaboration rules with other agents

Current active product development roles:
- BA Agent
- UXUI Agent
- FE Agent
- Reviewer Agent
- Research Agent
- Orchestrator Agent

The agent files are simple enough for BA, design, and delivery teams to read together.
