# Wireframe: Loyalty Overview Expansion

- UI ID: `UI-001`
- Parent FR ID: `FR-001`
- Parent FEAT ID: `FEAT-001`
## Primary Input
FRS from the BA Agent

## Screen Goal
Help the member understand points, tier progress, and recent loyalty activity in one place.

## Main Sections
- Header
- Member summary
- Points balance
- Tier progress
- Recent activity
- Help section

## Components
- Member overview card
- Available points panel
- Tier progress panel
- Recent transactions list
- Missing data guidance

## Interaction Notes
- Keep available points and expired points visually separate.
- Make tier progress easy to read at a glance.
- Show guidance when loyalty data is delayed.
