# Acceptance Criteria: Loyalty Overview Expansion

- AC ID: `AC-001`
- Parent Story ID: `US-001`
## Criteria
- Given member opens loyalty page, when data is available, then available points are shown.
- Given loyalty data is available, when page loads, then current tier and progress are shown.
- Given activity data is available, when member views page, then recent activity is displayed.
- Given loyalty data is missing or delayed, when page loads, then guidance message is displayed.
