# Wireframe: Ticket Booking Modification Improvement

- UI ID: `UI-001`
- Parent FR ID: `FR-001`
- Parent FEAT ID: `FEAT-001`
## Primary Input
FRS from the BA Agent

## Screen Goal
Help the customer understand booking change eligibility, fees, and next steps.

## Main Sections
- Header
- Booking summary
- Eligibility check
- Change options
- Fee review
- Help section

## Components
- Booking reference summary
- Eligibility banner
- Change options list
- Fee summary panel
- Support guidance message

## Interaction Notes
- Show eligibility status before deeper actions.
- Keep fee information visible before confirmation.
- Show support guidance when change is not allowed online.
