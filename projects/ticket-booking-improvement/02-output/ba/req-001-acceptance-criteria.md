# Requirement: req-001

# Acceptance Criteria: Ticket Booking Modification Improvement

- AC ID: `AC-001`
- Parent Story ID: `US-001`
## Criteria
- Given customer opens change page, when eligibility check runs, then system shows if booking can be changed online.
- Given booking is eligible, when options are displayed, then change fee is shown before confirmation.
- Given booking is not eligible, when customer opens page, then support/refund guidance is displayed.
- Given booking does not belong to customer, when they access page, then access is blocked.
