# Skills

Skills are reusable work patterns that agents can call.

Each skill explains:
- why it exists
- when to use it
- what input it expects
- what output it should produce
- the steps it follows
- the limits it should respect
- the expected Markdown outputs

Current skills now cover:
- BA outputs such as clarification, BPMN Mermaid, BRD, FRS, and feature breakdown
- UX/UI outputs such as wireframe guidance
- FE outputs such as simple HTML implementation

This makes the system easier to scale without rewriting the whole playbook each time.
