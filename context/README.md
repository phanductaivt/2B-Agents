# Context Pack

This folder stores reusable context so you do not need to repeat long setup text in every chat or run.

## Start Here
- Open `context/index.md` to see all files and what each file is for.
- Open `context/load-profiles.md` to choose a small context set by task type.

## Fast Usage
- Quick load: `context/context-summary.md`
- Deep load: `context/full-context.md`
- Task-focused load: pick one profile from `context/load-profiles.md`

## Maintenance
- Track updates in `context/change-log.md`
- Avoid common mistakes in `context/anti-patterns.md`
