# System Overview

## What This System Does
- Processes requirement files inside each project.
- Generates BA, UXUI, FE, review, test, and traceability outputs.
- Updates project status and global dashboard automatically.

## Key Capabilities
- Requirement clarification, BPMN, user story, AC, BRD, FRS, feature list.
- Wireframe generation and simple HTML output.
- Requirement and project traceability.
- Requirement ID system and linked artifact IDs.
- Dashboard views in Markdown and HTML.

## High-Level Flow
- `projects/<project-name>/inputs/requirements/*.md`
- BA Agent outputs
- UXUI Agent output
- FE Agent output
- Validators and review notes
- test cases and traceability files
- status and dashboard updates
